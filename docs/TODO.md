- [ ] 权限系统
    - [ ] 配置文件中默认 AI 的权限。包括“受限”“普通”“自由”和“极低”（这里，工作区权限覆盖全局权限，两个都可以通过配置文件调节）
        - [ ] 极低：AI 没有任何工具调用，即使是获取时间、阅读文件这样的。
        - [ ] 受限：AI 运行程序、操作命令行、写入/替换文件等之前需要用户批准，读取文件这一类不受影响。
        - [ ] 普通：AI 运行程序、操作命令行、写入/替换文件等完全自由，但如果触发了“权限越界异常”（例如使用读取文件的工具读取在配置中指定忽略的文件）就会降级到“受限”，需要用户手动将其调节回普通
        - [ ] 自由：AI 完全自由，就算触发了“权限越界异常”也不会降权
    - [ ] 应该在发生降权、AI 请求权限这些时机添加用户警告（这部分也许属于 GUI？）
